# ADR-001: Отказ от интерактивных CLI-агентов в пользу LangGraph

**Статус:** ACCEPTED  
**Дата:** 2024-06-16

## Контекст
Интерактивные TUI-агенты нестабильны в headless-режиме и склонны к зависаниям, что делает их непригодными для автономной работы.

## Решение
Использовать детерминированный LangGraph-оркестратор с явными State Machines (FSM) вместо вероятностных multi-agent блужданий.

## Последствия
Полный контроль над состоянием, возможность реализации self-healing циклов, предсказуемость.
