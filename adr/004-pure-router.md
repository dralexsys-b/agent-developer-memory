# ADR-004: Чистота Router-узлов в LangGraph

**Статус:** AGREED_DESIGN (Design agreed, implementation not verified)  
**Дата:** 2024-06-16

## Контекст
Смешивание логики маршрутизации и мутации состояния в одном узле приводит к сложным для отладки ошибкам.

## Решение
Согласовано, что Router-узлы должны быть чистыми функциями (возвращают только строку). Вся мутация состояния должна происходить строго в узлах обработки.

## Последствия
Дизайн утверждён. Реализация и проверка этого правила в коде оркестратора ещё не выполнены.
