# ADR-005: Lean Project Memory Architecture v1.0

**Статус:** ACCEPTED  
**Дата:** 2024-06-16

## Контекст
Чрезмерное усложнение системы документации создаёт неприемлемый оверхед для поддержки и ведёт к устареванию данных.

## Решение
Принята минимальная структура (Bootstrap v1.0): 5 файлов и 2 каталога (`adr/`, `evidence/`). Жёсткое правило: `CURRENT_CONTEXT.md` — единственный регулярно обновляемый файл.

## Последствия
Низкий оверхед на поддержку. Новая модель может восстановить контекст за секунды.
